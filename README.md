# SCT_DS_2_Data_Cleaning_and_Preparation
Data Cleaning and Preparation using Python and Pandas on the Sample Superstore Dataset. This project includes handling missing values, removing duplicate records, converting data types, and exporting the cleaned dataset.


# SkillCraft Technology Internship - Task 02

## 📌 Project Title
Data Cleaning and Preparation using Python & Pandas

## 📖 Project Overview
This project focuses on cleaning and preparing the Sample Superstore dataset for further analysis. The dataset was processed using Python and the Pandas library in Google Colab.

## 🎯 Objective
- Load the Sample Superstore dataset.
- Identify and handle missing values.
- Remove duplicate records.
- Convert date columns to datetime format.
- Export the cleaned dataset into a new CSV file.

## 🛠️ Technologies Used
- Python
- Pandas
- Google Colab
- Matplotlib

## 📂 Dataset
Sample Superstore Dataset

## 🔄 Data Cleaning Steps
- Loaded the dataset into Pandas.
- Checked for missing values.
- Filled missing values where required.
- Removed duplicate rows.
- Converted **Order Date** and **Ship Date** to datetime format.
- Saved the cleaned dataset as **Cleaned_SampleSuperstore.csv**.

## 📊 Data Visualization
The following visualizations were created:
- Sales by Category (Bar Chart)
- Profit by Category (Bar Chart)
- Sales Distribution (Histogram)
- Sales Share by Category (Pie Chart)
- Sales vs Profit (Scatter Plot)
